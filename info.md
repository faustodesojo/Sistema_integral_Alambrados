Prompt de Especificaciones: Sistema Integral Alambrados Belgrano
Contexto:
Necesito desarrollar un Sistema Integral de Gestión de Ventas y Logística para una empresa de cerramientos (Alambrados Belgrano). El sistema debe estar optimizado para venta en mostrador y gestión de inventario pesado. La interfaz debe seguir una estética de "Dark Mode" con acentos en naranja (según las imágenes de referencia).

Módulos y Funcionalidades Requeridas:

1. Cotizador Inteligente (Presupuestos)
Cálculo Dinámico: Capacidad de calcular materiales basados en:

Tipo de poste (Hormigón, madera, olímpico, etc.).

Tipo de tejido (Romboidal, electrosoldado) y su calibre.

Metros lineales totales y cantidad de "líneas" de cerramiento.

Lógica de Ángulos y Aberturas: El sistema debe calcular automáticamente los postes de refuerzo necesarios según la cantidad de ángulos (esquinas) y las aberturas (portones/puertas) ingresadas.

Accesorios: Selección de productos adicionales (alambre de púas, tensores, cemento, torniquetas).

Conversión: Botón para transformar un presupuesto aprobado en una "Orden de Venta" con un solo clic.

respetar los archivos con los calculos de los materiales NO MODIFICARLO

2. Gestión de Ventas y Cobranzas
Registro de Pagos: Soporte para múltiples métodos de pago (Efectivo, Transferencia, Tarjeta, Cuenta Corriente).

Pagos Parciales: Permitir señas para congelar stock/precio y saldos contra entrega.

Venta en Mostrador: Interfaz rápida de búsqueda de productos por código o nombre para ventas directas sin presupuesto previo.

3. Logística y Depósito (Gestión de Preparación)
Estados de Orden: Flujo de trabajo claro: Pendiente -> En Preparación -> Listo para Retiro/Envío -> Entregado.

Hojas de Carga: Generación de un "Plano de Trabajo" o "Lista de Picking" para que el personal de depósito sepa exactamente qué rollos y postes cargar.

Control de Stock: Descuento automático de inventario al marcar una orden como "En Preparación".

4. Devoluciones y Notas de Crédito
Reingreso a Stock: Al realizar una devolución, el sistema debe preguntar si el material vuelve al inventario (ej. rollos de tejido cerrados) o si se marca como scrap/dañado.

Gestión de Saldo: Generar un saldo a favor del cliente para futuras compras o realizar el egreso de caja correspondiente.

5. Finanzas y Caja Diaria
Apertura y Cierre: Registro de saldo inicial, ingresos por ventas y egresos por gastos menores.

Arqueo de Caja: Resumen detallado de cuánto debería haber en efectivo, tarjetas y bancos al finalizar el día.

Reportes en Tiempo Real: Visualización de ventas totales, ticket promedio y productos más vendidos (alambre vs. postes).

6. Administración y Configuración
Actualización Masiva de Precios: Herramienta para aumentar precios por porcentaje o por categoría (crucial para materiales de construcción).

Gestión de Clientes (CRM): Historial de presupuestos y compras por cliente para fidelización.

Requerimientos Técnicos Visuales:

UI/UX: Diseño limpio, tipografía legible, uso de tarjetas (cards) para separar secciones y estados representados por colores (Naranja para acciones, Verde para pagos, Azul para info).

Exportación: Capacidad de generar PDF profesionales de los presupuestos para enviar por WhatsApp o email.
